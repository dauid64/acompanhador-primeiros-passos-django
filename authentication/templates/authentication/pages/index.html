Ver sobre os capítulos

Capítulo 1 - Configurando ambiente

  • Instalar a Linguagem de Programação Python.
  • Entender e criar um ambiente virtual para desenvolvimento de aplicações Python.
  • Baixar e criar projeto Django.
  • Baixar o gerenciador de banco de dados PostgreSQL e criar um banco de dados

Ao final desse capítulo, você será capaz de configurar seu ambiente de desenvolvimento para trabalhar com Django, garantindo que todas as ferramentas necessárias estejam prontas para o desenvolvimento de aplicações web.

Capítulo 2 - Criando rotas

  • Criar as primeiras rotas no Django.
  • Renderizar nossas páginas com Template do Django.
  • Utilizar Class Based View para criar nossas views no Django

Neste capítulo, será introduzido o conceito de rotas no Django, permitindo que você crie URLs que respondem a requisições HTTP e renderizem páginas HTML usando os templates do Django. Você também aprenderá a usar as Class Based Views para estruturar suas views de forma mais organizada e reutilizável.

Capítulo 3 - Criando modelos

  • Conectaremos nossa aplicação Django com o Banco de Dados PostgreSQL.
  • Criaremos os modelos da nossa aplicação.
  • Discutiremos relacionamento de modelos no Django como Foreign Key e Many to Many

O aluno aqui será capaz de conectar sua aplicação Django ao banco de dados PostgreSQL, criando modelos que representam as entidades do seu sistema. Você aprenderá a definir campos, tipos de dados e relacionamentos entre modelos, como Foreign Key e Many to Many, para estruturar adequadamente os dados da sua aplicação.

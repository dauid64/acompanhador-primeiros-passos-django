Conheça os capítulos do curso

1 Configurando Ambiente

  • Instalar a Linguagem de Programação Python
  • Entender e criar um ambiente virtual para desenvolvimento de aplicações Python
  • Baixar e criar projeto Django
  • Baixar o PostgreSQL e criar um banco de dados
🎯 Objetivo: Ao final deste capítulo, você será capaz de configurar seu ambiente de desenvolvimento para trabalhar com Django, garantindo que todas as ferramentas necessárias estejam prontas para o desenvolvimento de aplicações web.

2 Criando Rotas

  • Criar as primeiras rotas no Django
  • Renderizar páginas com Templates do Django
  • Utilizar Class Based Views para criar views no Django
🎯 Objetivo: Neste capítulo, você será introduzido ao conceito de rotas no Django, permitindo criar URLs que respondem a requisições HTTP e renderizam páginas HTML usando templates. Aprenderá também a usar Class Based Views para estruturar suas views de forma organizada.

3 Criando Modelos

  • Conectar a aplicação Django com o Banco de Dados PostgreSQL
  • Criar os modelos da aplicação
  • Discutir relacionamentos como Foreign Key e Many to Many
🎯 Objetivo: Você será capaz de conectar sua aplicação Django ao PostgreSQL, criando modelos que representam as entidades do sistema. Aprenderá a definir campos, tipos de dados e relacionamentos entre modelos para estruturar adequadamente os dados.
